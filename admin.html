How this works / first-time setup
  1. This page is your private publishing tool. It isn't linked anywhere on your website, and it holds no secrets — publishing only works with your GitHub token, which is saved only in this browser.
  2. Create a token at github.com/settings/personal-access-tokens/new: give it a name like portfolio-admin, set Repository access to Only select repositories → sanjanataware.github.io, and under Permissions set Contents → Read and write. Copy the token and paste it below.
  3. Fill out the form, hit Publish, and the article is committed to your repo. GitHub Pages redeploys in about a minute.
  4. Content is written in Markdown: # Heading, **bold**, *italic*, [link text](url), ![caption](image-url), > quote, - list item.
